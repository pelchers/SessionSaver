# User Story Validation Template

## Story

- Title: Delete session
- Owner: product
- Role: As a user
- Goal: I want to delete sessions I no longer need
- Scope: MVP

## Steps

1) In the library, open a session row menu and click `Delete`.
2) Confirm deletion.

## Expected Results

- Confirmation is required before deletion.
- Session is removed from library and cannot be restored.
