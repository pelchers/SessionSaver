# User Story Validation Template

## Story

- Title: Restore full session
- Owner: product
- Role: As a user
- Goal: I want to restore a saved session to reopen all its windows/groups/tabs
- Scope: MVP

## Steps

1) Open a session detail view.
2) Click `Restore Full Session`.
3) Confirm restore in the dialog.

## Expected Results

- Confirmation dialog shows accurate counts (windows/tabs/groups).
- Restore opens the expected number of windows/tabs.
- Any skipped tabs are reported explicitly with reasons.
