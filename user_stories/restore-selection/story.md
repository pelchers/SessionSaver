# User Story Validation Template

## Story

- Title: Restore selection (window/group/tab)
- Owner: product
- Role: As a user
- Goal: I want to restore only part of a saved session so I can be selective
- Scope: MVP

## Steps

1) Open a session detail view.
2) Select a window node and restore it (confirm).
3) Select a group node and restore it (confirm).
4) Select a tab node and restore it (confirm).

## Expected Results

- Each restore operation scopes correctly (does not restore unrelated branches).
- Confirmation dialog counts match selected scope.
- Restore results are reported (success/partial/failure).
