# User Story Validation Template

## Story

- Title: Browse library
- Owner: product
- Role: As a user
- Goal: I want to browse sessions like files and sort/filter to find what I need
- Scope: MVP

## Steps

1) Open the extension library UI.
2) Change sort order (Updated, Name, Favorites first).
3) Toggle favorites-only filter on and off.

## Expected Results

- List updates deterministically with each sort selection.
- Favorites-only filter shows only starred sessions.
- Returning from a session detail view preserves current sort/filter state.
