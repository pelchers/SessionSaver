# Default Frontend Planning Styles

## `brutalist`
- High contrast
- Thick borders
- Loud type scale
- Deliberately raw layout blocks

## `code-terminal-ide`
- Dark workspace aesthetic
- Monospace typography
- Panel/window framing
- Status and command affordances

## `minimalist`
- Low-noise UI
- Neutral palette
- Generous whitespace
- Tight hierarchy with subtle accents

## `newspaper`
- Editorial layout and serif typography
- Column-based sections
- Headline and deck treatment
- Black/white with restrained accents

## `mid-century-modern`
- Warm retro tones
- Geometric motifs
- Strong typographic personality
- Clean but expressive composition
