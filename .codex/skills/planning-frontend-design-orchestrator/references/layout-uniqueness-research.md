# Layout Uniqueness Research Notes

Date: 2026-02-14

## Problem Statement
The Codex-based generation system produced 10 passes that all looked structurally identical because they were stamped from a single PowerShell template (`generate-concept.ps1`). The template rotated CSS variables (colors, fonts, border-radius) but kept the same:
- HTML structure (same shell, same nav, same card layout)
- Same card component shape (`.view-card` with slight class variations)
- Same nav button structure (`.nav-btn` with class variations)
- Same fixed 3D Three.js stage overlay on every pass
- Same downloaded background images from picsum.photos
- Same font mega-bundle (every pass loaded all fonts)

## Solution: Agent-Generated Concepts
Instead of template stamping, each concept pass is now generated by a Claude Code Task agent that:
1. Receives a unique creative brief
2. Writes ALL code from scratch
3. Makes its own structural decisions within the brief constraints
4. Uses style-appropriate animations (not forced Three.js on every pass)
5. Uses background images only when they serve the aesthetic

## Uniqueness Dimensions
Each pass should differ across these axes:
1. **Shell architecture**: Where does navigation live? How is the page split?
2. **Navigation pattern**: Tabs, sidebar, bottom bar, command palette, orbital, etc.
3. **Content flow**: Grid, masonry, stack, columns, horizontal scroll, etc.
4. **Typography system**: Not just different fonts - different sizing scales, line-heights, letter-spacing approaches
5. **Color application**: Same palette can look wildly different based on where colors are applied
6. **Spacing rhythm**: Tight vs airy, consistent vs varied
7. **Component language**: Sharp vs rounded, flat vs elevated, bordered vs borderless
8. **Interaction model**: How do views change? Slide? Fade? Snap? Cut?

## Validation
- `validate-design-uniqueness.mjs` performs pairwise HTML token + CSS variable comparison
- Threshold of 0.55 Jaccard similarity (same as before)
- Agent-generated concepts should naturally produce much lower similarity scores than template-stamped ones
