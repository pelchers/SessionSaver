# Store Listing Draft (Chrome Web Store)

## Name

SessionSaver

## Short Description

Save your current Chrome windows and tab groups as named sessions, then browse and restore them like files.

## Detailed Description

SessionSaver lets you capture your current browsing context as a “session” and come back later without rebuilding your tabs.

Key features:
- Save all open Chrome windows, tab groups, and tabs in one click
- Browse sessions in a Drive-like library
- Favorite sessions, add descriptions, and rename anytime
- Explore a session with a VS Code-style tree: windows -> groups -> tabs
- Restore everything or restore just what you need (window/group/tab)

Notes:
- Some internal or restricted pages may not be restorable due to Chrome limitations. SessionSaver will tell you what was skipped.

## Privacy Disclosure (Draft)

- Data is stored locally in your browser by default.
- No analytics/telemetry in v1.
- No data is sold or shared.

## Permissions Justification (Draft)

- Tabs: required to read open tabs and restore tabs.
- Tab Groups: required to save and recreate tab groups.
- Storage: required to save your sessions locally.
